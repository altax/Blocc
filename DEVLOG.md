# Twitch AI Bot — Development Log

## Цель проекта
Создать ИИ-бота для Twitch, который наблюдает за CS2-стримами (видео + аудио), читает чат и пишет сообщения, неотличимые от живого русского зрителя. Главная задача — стать популярным стримером, поэтому бот должен максимально точно имитировать поведение аудитории топовых русских CS2 стримеров.

---

## Принципы антидетекта
- Бот НЕ должен быть обнаружен как бот ни при каком условии
- Паттерны берутся исключительно из реального русского CS2 чата
- Задержки и частота сообщений имитируют реального зрителя (~раз в 1-3 мин)
- 15% шанс промолчать даже при активном триггере (как реальный человек)
- Симуляция опечаток у 18% сообщений (дублирование/пропуск букв)
- Инъекция реальных паттернов в 30% сообщений
- Cooldown рандомизирован, рекомендуется 60–120с

## Архитектурные решения
- **Паттерны в БД с приоритизацией** — score = (ru×3) + (cs2×2) + frequency. Русские CS2 паттерны всегда в топе.
- **Инъекция паттернов в промпт** — топ-20 реальных фраз инжектятся в системный промпт GPT-4o-mini при каждой генерации. Это ключевой механизм имитации.
- **Anti-detect через рандомизацию** — задержки, вероятностное молчание, редкие опечатки. Фиксированные паттерны сразу видны модераторам.
- **IRC anonymous для обучения** — при сборе паттернов используем justinfanXXXXX, никакого риска бана аккаунта.
- **Bulk learn последовательно** — каналы обучаются по очереди с паузой 2с между ними, чтобы не получить ban от Twitch за спам подключений.
