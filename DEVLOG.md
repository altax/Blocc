# Twitch AI Bot — Development Log

## Цель проекта
Создать ИИ-бота для Twitch, который наблюдает за CS2-стримами (видео + аудио), читает чат и пишет сообщения, неотличимые от живого русского зрителя. Главная задача — стать популярным стримером, поэтому бот должен максимально точно имитировать поведение аудитории топовых русских CS2 стримеров.

---

## Принципы антидетекта
- Бот НЕ должен быть обнаружен как бот ни при каком условии
- Паттерны берутся исключительно из реального русского CS2 чата
- Задержки и частота сообщений имитируют реального зрителя (~раз в 1-3 мин)
- 15% шанс промолчать даже при активном триггере (как реальный человек)
- Симуляция опечаток у 18% сообщений (дублирование/пропуск букв)
- Инъекция реальных паттернов в 30% сообщений
- Cooldown рандомизирован, рекомендуется 60–120с

### [2026-06-01] — GQL multi-alias, улучшенный классификатор, Тест ИИ, редизайн Обучения ✅
**Что сделано:**

**1. GQL online-check (game-detector.ts) — полная перезапись:**
- Batch-array → единый запрос с множественными alias: `{ ch0: user(login:"x"){...} ch1: ... }`
- Ответ: `{ data: { ch0: {...}, ch1: {...} } }` — надёжнее, не попадает под rate-limit batch-mode
- `AbortSignal.timeout(15_000)` — защита от зависания
- Логирует `online: "ch(игра), ..."` в одной строке — удобнее для отладки

**2. Классификатор паттернов (pattern-learner.ts) — значительное улучшение:**
- Два новых типа: `cs2_reaction` (эмоциональные крики/возгласы: ору, красавчик, вау) и `cs2_result` (итог раунда: gg, ez, слили, разнесли)

- `detectGame()` — расширенный словарь: deagle, флагман, феник, дефуз, пистолетный и др.
- `isHumanLike()` — новые фильтры: повторяющиеся символы (5+), эмодзи-спам, follow/sub/raid сообщения

- Активные записи выведены в начало страницы с развёртываемым live-чатом (авто-открывается при старте сессии)
- `LiveChatFeed` — живой скролл сообщений с IRC, обновляется каждые 2с, авто-скролл + кнопка "к концу"
- Секции разделены: CS2 онлайн → Другие онлайн → Оффлайн — без дублирования
- Нижняя строка статуса планировщика

**Что сделано — Паттерны:**
- Двухпанельный layout: левая панель = источники обучения (стримеры из API), правая = база паттернов
- Стримеры берутся из `useGetStreamerPresets()` — один источник правды со страницей стримеров
- Показывается кол-во паттернов от каждого стримера прямо в строке
- **ТОП-20 паттернов** которые ИИ реально вставляет в сообщения — с пронумерованными бейджами
- Статистика по типам паттернов как кликабельные фильтры (CS2 каллаут, рус. сленг, хайп...)
- `FrequencyBar` — визуальная шкала частоты для каждого паттерна
- Паттерны с номером `rank` (1–20) помечены как "используется ИИ" фиолетовым кружком
- Поиск по тексту паттерна, фильтр по каналу (клик на стримера = фильтр справа)
- Нижняя строка: как именно ИИ использует паттерны (30% шанс, 18% опечатки, 15% молчание)
- Авто-обновление данных каждые 10 секунд

**Результат:** обе страницы полностью переработаны, статус определяется автоматически, чат активности виден в реальном времени, процесс обучения ИИ прозрачен.

## Архитектурные решения
- **Паттерны в БД с приоритизацией** — score = (ru×3) + (cs2×2) + frequency. Русские CS2 паттерны всегда в топе.
- **Инъекция паттернов в промпт** — топ-20 реальных фраз инжектятся в системный промпт GPT-4o-mini при каждой генерации. Это ключевой механизм имитации.
- **Anti-detect через рандомизацию** — задержки, вероятностное молчание, редкие опечатки. Фиксированные паттерны сразу видны модераторам.
- **IRC anonymous для обучения** — при сборе паттернов используем justinfanXXXXX, никакого риска бана аккаунта.
- **Bulk learn последовательно** — каналы обучаются по очереди с паузой 2с между ними, чтобы не получить ban от Twitch за спам подключений.
